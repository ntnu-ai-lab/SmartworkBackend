This file explains how to start the SmartWork backend application.
		
*************************************************************************************************************
 Caution: Below information is for the administrator of the backend services and not meant for general users!
*************************************************************************************************************


(o) Start backend services:	run the command: nohup ./start-backend.sh &
	This command will start all the required backend systems (it may take up to few minutes to get everything running).

(0) open the link: http://smartwork.idi.ntnu.no:20000/swagger-ui.html#/

Note:  Below command will stop all the backend services except the elasticsearch

(o) Stop backend services: run the command: ./stop-backend.sh
